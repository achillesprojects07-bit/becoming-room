# The Becoming Room — V1.0

A first working PWA prototype for a private clarity, self-trust, evidence-based rewiring, and grounded manifestation journal.

## What is included
- Daily check-in: feeling → what it reveals → desire → aligned action.
- Trigger mode: fact vs meaning → body signal → urge → regulation → 10-minute test → evidence.
- Journal Page mode: real journal page look, optional photo, paper styles, borders, floral elements, cursive-style display.
- Entries history saved locally in the browser.
- Becoming mode: future self direction, evidence bank, repeated patterns.
- Small visible version label: V1.0.

## Important note
This version saves to local browser storage only. It is private on your device/browser, but it is not yet synced across devices. Firebase can be added later for login, backup, and photo storage.

## How to test
Open `index.html` in a browser. For mobile testing, upload the folder to GitHub Pages, Vercel, Netlify, or Firebase Hosting.
